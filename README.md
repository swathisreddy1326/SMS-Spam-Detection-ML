# SMS Spam Detection using Machine Learning
This project detects whether a given SMS message is Spam or Not Spam using Machine Learning and Natural Language Processing.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (CountVectorizer)

## Dataset
The dataset contains 5572 SMS messages labeled as:
- Ham (Not Spam)
- Spam

## Machine Learning Model
We used Multinomial Naive Bayes classifier for text classification.

## Steps Performed
1. Data Cleaning
2. Label Encoding
3. Text Vectorization using CountVectorizer
4. Train Test Split
5. Model Training
6. Prediction

## Accuracy
Model Accuracy: **97.84%**

## Example Prediction
Input: "Congratulations! You have won a free lottery prize"
Output: Spam Message

Input: "Hey, did you complete the assignment?"
Output: Not Spam Message
